# Lawyer Consultency API

API for lawyer consultency form

## Installation

Use the package manager [yarn](https://classic.yarnpkg.com/lang/en/docs/install) to install this project.
Create database using schema `assets/sql.sql`

```bash
yarn
```

## Run Project

```bash
yarn start
```
